# GEMINI CLI Workflow Guidelines

## Change Logging
*   **Central Log**: All changes made to this repository must be logged in the `GEMINICLI_changelog.txt` file located in the root directory.
*   **Format**: Entries should be clear and concise.
*   **Paths**: Always use relative paths (e.g., `./folder/file.ext`) when referencing modified files.
*   **Content**: Explain *what* changed and *why*.

## Project Structure
*   **Root**: Contains project-wide configuration and logs.
*   **Labs**: Individual lab assignments are in `labX/` folders.
